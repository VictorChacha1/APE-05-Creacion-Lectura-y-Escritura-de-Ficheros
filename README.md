## 🎯 Objetivos del Proyecto
* Manipular archivos mediante métodos y clases que proporciona el lenguaje de programación Java.
* Implementar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) simuladas en ficheros de texto plano secuenciales.
* Desarrollar validaciones estrictas y menús interactivos garantizando la integridad de los datos ingresados por el usuario.

---

## 🚀 Listado de Ejercicios Implementados

El repositorio cuenta con 10 ejercicios funcionales:

1. **Registro de Estudiantes:** Escritura en modo *append* de datos académicos.
2. **Lectura de Archivo de Estudiantes:** Extracción, iteración y cálculo matemático de promedios leyendo un `.txt`.
3. **Sistema de Inventario Básico:** Registro de productos y cálculo del valor total del inventario.
4. **Agenda Telefónica:** Algoritmo de búsqueda de subcadenas para localizar contactos guardados.
5. **Sistema de Notas:** Evaluación condicional encapsulada para definir el estado (Aprobado/Reprobado).
6. **Control de Asistencia:** Lógica de calendario estricta (5 días laborables continuos) para conteo de faltas.
7. **Cajero Automático Básico:** Lectura, cálculo y sobrescritura de saldo.
8. **Biblioteca Virtual:** Filtrado de líneas mediante el estado de los libros (Disponibles/Prestados).
9. **Sistema de Ventas:** Formateo de texto avanzado para la generación de facturas persistentes.
10. **Mini Proyecto Integrador (CRUD):** Algoritmo de borrado de registros utilizando archivos temporales (`temp.txt`) y renombrado de ficheros.
